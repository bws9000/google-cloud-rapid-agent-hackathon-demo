StateFlowX Runtime Demo

Getting Started

To run this demo, enter your Gemini API key and click Execute Task.

Your API key is used only for the current execution and is not stored by StateFlowX. If you prefer, use a temporary or limited-access key for testing.

What This Demo Does

This demo executes a StateFlowX workflow and streams execution events to the dashboard in real time.

  1. Calls a weather service to retrieve live weather data.
  2. Passes the response into an AI-powered workflow.
  3. Uses Google ADK and MCP integration to access external tools.
  4. Generates a structured JSON result.
  5. Emits runtime events as the workflow progresses.

Configuration Example


defineConfig({{ '{' }}
  transport: websocket(),
  protocol: jsonRpc(),

  services: [
    {{ '{' }}
      name: 'weather',
      type: 'http'
    {{ '}' }}
  ],

  workflows: [
    {{ '{' }}
      route: 'weather.execute',
      service: 'weather',
      provider: 'google-adk'
    {{ '}' }}
  ]
{{ '}' }});

Results

The Result column contains the final workflow output. In this demo the output is JSON, but a workflow could produce any type of artifact, including documents, reports, images, generated code, or custom application data.
